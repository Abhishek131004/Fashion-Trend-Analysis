#  Fashion Trend Analysis

A comprehensive **Exploratory Data Analysis (EDA)** project that analyzes fashion products to uncover trends in brands, categories, pricing, discounts, and customer preferences. Using Python and data visualization techniques, this project transforms raw fashion retail data into meaningful business insights that can help retailers understand market trends and make data-driven decisions.

---

#  Project Overview

The fashion industry is highly dynamic, with changing customer preferences, seasonal trends, and pricing strategies. This project explores a fashion product dataset to identify patterns in product distribution, pricing, discounts, brands, and gender-specific preferences.

The analysis provides valuable insights into:

- Most popular brands and product categories
- Price segment distribution
- Product popularity across genders
- Category performance across different price ranges
- Brand dominance in each price segment
- Discount strategies across categories
- Best-selling products by category
- Interactive filtering for customized analysis

---

#  Key Insights

- **Performax** emerged as the most represented brand in the dataset.
- **Topwear** and **Westernwear** dominate the overall product distribution.
- **Crew-Neck T-Shirts**, **Straight Kurtas**, and **Fit Shirts** are among the most popular products.
- The **Mid Price Segment (₹500–₹1499)** contains the largest number of products, indicating strong demand for affordable fashion.
- **Topwear** dominates the mid-price segment, while **Footwear** maintains strong representation across multiple price ranges.
- **Deep Discounts (>40%)** are the most common promotional strategy adopted by brands.
- Men's fashion is largely driven by **Fit Shirts**, **Crew-Neck T-Shirts**, and **Polo T-Shirts**.
- Women's fashion is dominated by **Straight Kurtas**, **A-line Kurtas**, and **Crew-Neck T-Shirts**.
- Interactive filtering allows users to analyze products based on multiple attributes for more personalized insights.
